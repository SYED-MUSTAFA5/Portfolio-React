# Protfolio Project

A modern, high-performance web application built with **React** and **Vite**. This project focuses on speed, clean code, and a smooth user experience.

---

## ⚡ Features

* **Fast Refresh:** Powered by Vite for near-instant hot module replacement (HMR).
* **Responsive UI:** Optimized for all screen sizes.
* **Modern React:** Built using Functional Components and Hooks.
